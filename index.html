Targét

This is a Target inspired web app with a new twist, it is called Targét. It's got the same stuff as Target but it's more expensive! Using a flask/sqlalchemy backend and a react.js/redux.js frontend I was able to implement 3 functional features in just under 2 weeks including planning. Currently, you are able to add items to your cart or wishlist and also review items present. More features coming soon! Including a product search~.

Not OK, Cupid

Not OK, Cupid is an OK Cupid inspired web application. Our site can be used to create a user, view current users, message mismatches(people whom you have disliked and have also disliked you back), answer some cheeky personality questions, and just enjoy yourself while browsing the page.

Meet, Sup?

Meet, Sup? is a Meetup clone where you can make groups and events. It uses a sequelize/express backend paired with a react.js/redux.js frontend to deliver a smooth user experience. Feel free to sign up and take a spin, or just login as the demo user.
